# Cabo Server

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
# install dependencies
$ pnpm install

# deploy migrations
$ pnpm prisma migrate deploy

# generate Prisma client
$ pnpm prisma generate
```

## Compile and run the project

```bash
# development
$ pnpm start

# watch mode
$ pnpm start:dev

# production mode
$ pnpm start:prod
```
